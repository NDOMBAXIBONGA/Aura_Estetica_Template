1. Introdução

O Aura Estética é um template estático premium de página única (landing page), responsivo e de alto desempenho, especialmente projetado para clínicas de estética, spas médicos, estúdios de harmonização e profissionais de beleza de luxo.

Com foco absoluto em conversão de visitantes para agendamentos, o template traz uma paleta de cores sofisticada em tons terra e ouro, tipografia refinada e um Widget de Agendamento Interativo de 4 etapas no estilo SPA (Single Page Application) com animações fluidas geradas pela biblioteca GSAP.

2. Estrutura de Arquivos

A organização do template é limpa e direta, facilitando a edição e personalização em qualquer IDE.

/ ├── index.html # Estrutura HTML5 semântica principal e tags SEO ├── style.css # Estilos CSS complementares (gradients, glassmorphism) ├── script.js # Lógica interativa (Widget, FAQ Accordion, GSAP) ├── readme.txt # Instruções rápidas do produto ├── documentation/ # Pasta com a documentação do template │ └── index.html # Esta página de documentação └── assets/ # Recursos de imagem e logotipos ├── logo-icon.png # Ícone do logotipo da clínica ├── hero-clinic.png # Imagem da seção de boas-vindas ├── treatment-*.png # Imagens dos tratamentos oferecidos ├── professional-*.png # Fotos da equipe profissional └── avatar-client-*.png # Avatares dos clientes em depoimentos

4. Como Executar o Template

Como este template é 100% estático, não há necessidade de compiladores, Node.js ou bancos de dados locais.

Execução Simples

Dê dois cliques no arquivo index.html para abri-lo diretamente no Chrome, Firefox, Safari ou Edge.

Servidor de Desenvolvimento (Recomendado)

Para testar animações GSAP de forma ideal, use a extensão "Live Server" do VS Code ou execute via Python no terminal da pasta do projeto:

python -m http.server 8000

Em seguida, acesse http://localhost:8000 no navegador.

5. Como Customizar

A. Alterar Cores

O projeto possui cores sofisticadas configuradas diretamente no Tailwind CSS dentro do index.html e variáveis CSS em style.css.

/* No index.html, altere em tailwind.config: */ colors: { sand: { DEFAULT: '#FAF7F5', // Fundo principal claro dark: '#F4EFEA', // Fundo secundário }, champagne: '#D4AF37', // Ouro vibrante goldrose: '#C5A880', // Ouro Rose luxo sage: { DEFAULT: '#6E8A78', // Verde sálvia relaxante } }

Lembre-se de atualizar também os valores hexadecimais no `:root` em style.css para manter o widget e elementos extras sincronizados.

B. Mudar Serviços e Equipe

Os tratamentos e especialistas aparecem tanto na landing page quanto nas opções de clique do widget.

  1. No index.html, altere o atributo data-value nos botões de escolha no Widget de agendamento (Passo 1 e 2).
  2. No script.js, altere as tabelas de mapeamento tratamentosNomes e profissionaisNomes nas linhas ~310 a ~323 para que o resumo final exiba as novas descrições corretamente.

6. Conectar o Agendamento ao WhatsApp

Como este é um template estático, por padrão o Passo 4 mostra apenas uma tela de simulação bem-sucedida. Se você quiser enviar as informações inseridas pelo usuário para o WhatsApp da clínica, adicione o seguinte trecho ao evento do botão de conclusão (next-step-3) no script.js:

// Substitua o evento click de 'next-step-3' no seu script.js (Linha ~345): document.getElementById('next-step-3').addEventListener('click', () => { if (validateStep3()) { bookingState.clientName = inputName.value.trim(); bookingState.clientEmail = inputEmail.value.trim(); bookingState.clientPhone = inputPhone.value.trim(); // Gerar link e abrir WhatsApp const foneClinica = "5521988887777"; // Insira o número completo com DDD (e 55 para Brasil) const mensagem = `Olá! Gostaria de confirmar meu agendamento na Clínica Aura: - *Tratamento:* ${bookingState.treatment} - *Profissional:* ${bookingState.professional} - *Data e Hora:* ${bookingState.date} às ${bookingState.time} - *Nome Paciente:* ${bookingState.clientName} - *Telefone:* ${bookingState.clientPhone}`; const urlWhatsapp = `https://api.whatsapp.com/send?phone=${foneClinica}&text=${encodeURIComponent(mensagem)}`; window.open(urlWhatsapp, '_blank'); goToStep(4); } });

Dessa forma, após finalizar o formulário, o cliente é direcionado instantaneamente para iniciar a conversa já enviando os dados da reserva.

7. Hospedagem & Deploy

Para publicar o seu site na internet e obter o link de demonstração ao vivo para colocar no Codester, você pode hospedar o template gratuitamente nas seguintes plataformas:

GitHub Pages (Fácil & Gratuito)

Se você usa o GitHub para versionamento de código, pode publicar em poucos cliques:

  1. Crie um repositório público e envie os arquivos do projeto.
  2. Acesse Settings > Pages no seu repositório.
  3. Em "Build and deployment", selecione a branch main e a pasta / (root).
  4. Clique em salvar. Em instantes o site estará online no link fornecido.
Vercel ou Netlify (Arrastar e Soltar)

Se você não utiliza terminal ou Git, basta arrastar a pasta do projeto diretamente nas telas dessas plataformas para gerar um link de produção instantaneamente com certificado SSL de graça.
